## Whatsapp Gila

Gunakan tools ini untuk kebutuhan whatsapp

## Featured

- Save nomor tidak kamu save. ( melalui status yg dibuat teman mu)
- Save nomor orang yg gak read nomor kamu :'(
- Kirim chat

## Cara install

- buka cmd / terminal

- kemudian ketik

`git clone https://github.com/mrtampan/whatsapp-gila.git`

- lalu enter

- ketik lagi

`npm install`

- lalu enter

- ketik

`node index.js`

- scan whatsapp mu

- selesai
